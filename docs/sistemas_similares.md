# Sistemas Similares - Referências para o InsuGuia Mobile

## 1. InsulinAPP (Brasil)

**URL:** https://www.insulinapp.com.br/hotsite/

**Descrição:** Solução hospitalar inteligente para controle glicêmico desenvolvida no Brasil.

**Funcionalidades Principais:**

### Avaliação Inicial
Introdução do paciente ao protocolo de hiperglicemia hospitalar, gerando uma prescrição completa de insulinoterapia baseada nos dados clínicos e laboratoriais.

### Seguimento
Auxílio na evolução do paciente através de ajuste de doses de insulina para a nova prescrição médica, de acordo com as glicemias capilares prévias.

### Plano de Alta Hospitalar
Ajuda na decisão de orientações para a alta hospitalar do paciente com diabetes mellitus ou hiperglicemia hospitalar.

### UTI
Auxílio no manejo da transição da insulinoterapia endovenosa em bomba de infusão contínua para dose subcutânea de insulina.

**Plataformas:**
- Aplicativo móvel (Android/iOS)
- Versão web para navegador desktop

**Instituições que utilizam:**
- InCor
- Hospital Sírio Libanês
- Outras instituições hospitalares brasileiras

**Nota:** Conforme mencionado pelo Dr. Itairan, este sistema existe mas não está adequado às atuais diretrizes da SBD 2025.

## 2. Calculadora de Insulina - Hospital Albert Einstein

**URL:** https://medicalsuite.einstein.br/Servicos/Paginas/calculadora_Insulina.aspx

**Descrição:** Ferramenta online para cálculo de doses de insulina hospitalar.

**Características:**
- Requer 2 glicemias acima de 180 mg/dL para inclusão no protocolo
- Baseada em dados clínicos e laboratoriais do paciente
- Utilizada internamente no Hospital Albert Einstein

## 3. Sistemas Internacionais (Referências nas Diretrizes SBD)

Conforme mencionado nas diretrizes da Sociedade Brasileira de Diabetes, existem sistemas similares em língua inglesa que podem servir como norteadores para o desenvolvimento do InsuGuia Mobile.

## Diferenciais do InsuGuia Mobile

O InsuGuia Mobile será desenvolvido com as seguintes características diferenciadoras:

1. **Atualizado com Diretrizes SBD 2025:** Seguirá rigorosamente as diretrizes mais recentes da Sociedade Brasileira de Diabetes.

2. **Foco em Paciente Não Crítico:** Implementação completa e detalhada para o cenário de paciente não crítico, não gestante.

3. **Categorização de Sensibilidade Insulínica:** Sistema robusto para classificar pacientes como sensíveis, usuais ou resistentes à insulina.

4. **Opções de Insulina Disponíveis no SUS:** Priorização de NPH e Regular, mais amplamente disponíveis nos hospitais brasileiros.

5. **Interface Intuitiva:** Design focado na experiência do médico durante o atendimento hospitalar.

6. **Prescrição Completa:** Geração de prescrição detalhada incluindo dieta, monitorização, insulinoterapia e manejo de hipoglicemia.

7. **Código Aberto e Acadêmico:** Projeto desenvolvido como extensão universitária, com potencial para evolução contínua.

## Aprendizados dos Sistemas Existentes

### Estrutura de Fluxo
- Cadastro inicial do paciente com dados clínicos
- Avaliação e categorização
- Geração de prescrição inicial
- Acompanhamento diário com ajustes
- Orientações para alta hospitalar

### Dados Necessários
- Identificação do paciente
- Dados antropométricos (peso, altura, IMC)
- Dados laboratoriais (creatinina, glicemias)
- Histórico de diabetes e uso prévio de insulina
- Tipo de dieta recebida
- Uso de medicações que afetam glicemia (corticoides)

### Interface
- Simplicidade e praticidade
- Disponibilidade em múltiplas plataformas
- Cálculos automáticos com transparência nas fórmulas
- Avisos e alertas clínicos importantes
